# Documenting Solar Eclipse Photography - Version 3.1.0

## Updates in Version 3.1.0

### New Features:
- Monitoring server implemented to track reflections.
- Enhanced main controller for improved performance.
- New dependencies added:
  - PyYAML
  - colorlog
  - python-dateutil
  - pyserial

### Changes:
- Updated requirements for the new dependencies.
- Implemented signal handling for better control.
- Features for graceful shutdown added, ensuring clean exit during operation.

## Setup Instructions
Ensure you install required dependencies by running:
```
pip install -r requirements.txt
```
